# Security / Securite

[English](#english) | [Francais](#francais)

## English

The SDK is a client package. Most sensitive behavior is enforced by Engine, but
client code still handles URLs, JWTs, and request bodies.

Report security issues privately to the maintainers before opening a public
issue. Include:

- affected SDK version or commit
- affected Engine endpoint, if known
- reproduction steps
- impact
- suggested fix, if available

Do not include real JWTs, customer data, payment references, phone numbers, or
addresses in public issues.

## Francais

Le SDK est un package client. La plupart des controles sensibles sont appliques
par Engine, mais le code client manipule quand meme les URLs, JWTs et corps de
requete.

Signalez les problemes de securite en prive aux maintainers avant d'ouvrir une
issue publique. Ajoutez:

- version SDK ou commit concerne
- endpoint Engine concerne, si connu
- etapes de reproduction
- impact
- correction suggeree, si disponible

N'incluez pas de vrais JWTs, donnees client, references paiement, numeros de
telephone ou adresses dans les issues publiques.
